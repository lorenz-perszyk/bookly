# Bookly App

## About
The Bookly App utilizes the Google Books API.<br/>
Lets you search by keyword, and several optional parameter.<br/>
You can store your favorites in the bookshelf.


## Tech
- React
- Emotion/Styled Components
- TypeScript
- Vite
